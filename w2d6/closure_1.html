Does a function pickup latest changes?
The function sayHi uses an external variable name. When the function runs, which value is it going to use?

let name = "John";
function sayHi() {
alert("Hi, " + name);
}

name = "Pete";

sayHi();

"Pete" is the output value
Because the function uses the most recent values

Are counters independent?
Here we make two counters: counter and counter2 using the same makeCounter function.
Are they independent? What is the second counter going to show? 0,1 or 2,3 or something else?

function makeCounter() {
let count = 0;

return function () {
return count++;
};
}

let counter = makeCounter();
let counter2 = makeCounter();

alert(counter()); // 0
alert(counter()); // 1

alert(counter2()); // ?
alert(counter2()); // ?
The output values of counter2 are 0 and 1. Counter and counter2 are independent.
They have independent outer Lexical Environments

Counter object
Here a counter object is made with the help of the constructor function.
Will it work? What will it show?

function Counter() {
let count = 0;

this.up = function () {
return ++count;
};
this.down = function () {
return --count;
};
}

let counter = new Counter();

alert(counter.up()); // ?
alert(counter.up()); // ?
alert(counter.down()); // ?

It works
The output values are 1, 2, 1
